# Challenge A

Given a **sorted** integer array of size `n`,
where each value is either duplicated once or unique,
find the unique value and return it.

The solution should be an optimized solution.

## Examples

**Input:**
`[-1, -1, 3, 3, 4, 5, 5, 9, 9]`

**Output:**
`4`

---

**Input:**
`[-6, -5, -5, -2, -2, 14, 14, 20, 20, 21, 21, 82, 82, 1000, 1000, 1450, 1450]`

**Output:**
`-6`
